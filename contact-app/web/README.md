### Tech Stack 🛠️

- React
- Styled Components
- React Router
- ESLint
- Prettier
- Vite

## Installation 💻

> classic intsallation for any rect project (swap bun with npm if needed)

1. Clone the repo
2. Install dependencies
3. Run the app
4. Build the app

```bash
git clone https://github.com/gabriel-lutz/my-contacts.git
cd my-contacts
```

```bash
bun install
```

## Usage 📚

> run the app

```bash
bun run dev
```

> build the app

```bash
bun run build
```
